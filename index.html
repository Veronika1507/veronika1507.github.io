CSS selectors

CSS selectors Example of selector Description
.class .introduction Selects all elements with class="intro"
#id #white Selects the element with id="white"
* * Selects all elements
element a Selects all <a> elements
element.class p.introduction Selects all <p> elements with class="introduction"
element,element h1,div Selects all <h1> and all <div> elements
element element div p Selects all <p> elements inside <div> elements
element>element div > p Selects all <p> elements where the parent is a <div> element
element+element div + p Selects the first <p> element that is placed immediately after <div> elements
element1 ~ element2 h1 ~ p Selects every first <p> element that is preceded by a <h1> element
